# GSC MCP — Sample Prompts for Claude Desktop

Copy any of these into Claude Desktop once your GSC MCP is connected.

---

## 🔌 Connection Test

```
List all the Search Console properties I have access to.
```

---

## 📊 Performance & Traffic

```
Show me the top 20 pages by clicks on [yoursite.com] for the last 30 days.
Include impressions, CTR, and average position for each.
```

```
What are the top 10 search queries driving traffic to [yoursite.com] this month?
```

```
Compare my traffic on [yoursite.com] from last month vs the month before.
Which pages improved and which dropped?
```

```
Show me all pages on [yoursite.com] where CTR is below 2% but impressions are above 200.
These are pages where my titles and descriptions need work.
```

---

## 🔍 Indexing & Coverage

```
List all pages on [yoursite.com] with indexing errors.
Group them by error type and tell me which are most urgent to fix.
```

```
Inspect the URL [yoursite.com/page-name] and tell me:
- Is it indexed?
- When was it last crawled?
- Are there any issues?
```

```
Which pages on [yoursite.com] are excluded from the index and why?
```

---

## ⚡ Quick Wins

```
Show me all pages on [yoursite.com] with more than 50 impressions
sitting between position 8 and 15.
These are my quick-win opportunities for getting to page one.
```

```
Which search queries is [yoursite.com] almost ranking on page one for?
Show me queries with average position between 8 and 12.
```

---

## 🗺️ Sitemaps

```
What sitemaps are currently submitted for [yoursite.com]?
Are any returning errors?
```

```
Delete the broken sitemap at [yoursite.com/old-sitemap.xml].
```

```
Submit [yoursite.com/sitemap.xml] and confirm it was accepted.
```

---

## 📋 Client Reporting

```
Pull a full monthly SEO report for [yoursite.com] covering:
- Top 10 pages by clicks
- Top 10 search queries
- Any indexing errors
- Sitemap status
- Pages sitting between position 8-15 as quick wins
Format it clearly so I can share it with my client.
```

---

## 📱 Technical Checks

```
Are there any mobile usability issues on [yoursite.com]?
```

```
Check the structured data / rich results status for [yoursite.com].
Are there any errors I need to fix?
```

```
Show me the current crawl errors on [yoursite.com].
```
